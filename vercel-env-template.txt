REACT_APP_FIREBASE_AUTH_DOMAIN=your_project_id.firebaseapp.com

# Instructions:

# 2. Replace 'your_project_id' with your actual Firebase project ID
# 3. Replace 'your_messaging_sender_id' with your actual messaging sender ID
# 4. Replace 'your_app_id' with your actual Firebase app ID
# 5. Copy each line (without the # comments) and paste into Vercel
# 6. Set Environment to "Production" for all variables
# 7. Click "Save" after adding each variable
